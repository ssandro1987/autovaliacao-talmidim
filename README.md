# Autoavaliação Evangelização Discipuladora

Aplicação web simples em HTML, CSS e JavaScript para autoavaliação do pilar Evangelização Discipuladora.

## Funcionalidades

- Campo para informar o nome
- Cinco perguntas com respostas de 0 a 3
- Exibição da escala de pontuação
- Cálculo da pontuação total de 0 a 15
- Resultado exibido na mesma página
- Análise conforme a faixa de pontuação
- Gráfico radar com 5 eixos usando Chart.js
- Layout limpo, responsivo e em português do Brasil

## Faixas de análise

- **0 a 5:** Identificamos que este pilar ainda não faz parte do seu caminhar como discípulo. Existe uma fragilidade que pede cuidado imediato e uma escolha prática de mudança.

- **6 a 10:** Você já deu os primeiros passos e tem consciência dessa necessidade, mas falta constância. Para avançar, será preciso agir com mais intenção e disciplina diária.

- **11 a 15:** Parabéns, esta área já floresce em sua vida cristã. O próximo passo é manter o que foi conquistado, aprofundar suas raízes e proteger esse hábito com zelo.

## Como usar

Abra o arquivo `index.html` em qualquer navegador moderno.

Também é possível publicar esta aplicação em serviços como:

- GitHub Pages
- Netlify
- Vercel
- Google Sites, usando incorporação por código ou URL

## Observação

Esta versão não envia e-mails, não salva respostas e não utiliza banco de dados. Todo o cálculo é feito diretamente no navegador da pessoa.
